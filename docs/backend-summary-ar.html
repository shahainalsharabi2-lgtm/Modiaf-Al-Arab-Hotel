*Modiaf.Al.Arab.Hotel — الإصدار v2.3.3 — ASP.NET Core + ABP Framework 10*

1. نظرة معمارية

الخادم مبني على Domain Driven Design (DDD) ضمن ABP Framework 10، ويتكون من طبقات منفصلة:


HttpApi.Host  →  HttpApi  →  Application  →  Domain  ←  EntityFrameworkCore
                    ↑              ↑
            Application.Contracts  Domain.Shared
الطبقةالمشروعالدور
Domain.SharedModiaf.Al.Arab.Hotel.Domain.Sharedثوابت، أدوار، توطين ABP
DomainModiaf.Al.Arab.Hotel.Domainكيانات، مستودعات (واجهات)، بذور البيانات
Application.ContractsModiaf.Al.Arab.Hotel.Application.ContractsDTOs، واجهات App Services
ApplicationModiaf.Al.Arab.Hotel.Applicationمنطق التطبيق، App Services
EntityFrameworkCoreModiaf.Al.Arab.Hotel.EntityFrameworkCoreDbContext، Repositories، Migrations
HttpApiModiaf.Al.Arab.Hotel.HttpApiControllers إضافية
HttpApi.HostModiaf.Al.Arab.Hotel.HttpApi.Hostنقطة التشغيل، CORS، Swagger، OpenIddict
HttpApi.ClientModiaf.Al.Arab.Hotel.HttpApi.Clientعميل API للاختبار
DbMigratorModiaf.Al.Arab.Hotel.DbMigratorترحيل قاعدة البيانات

البنية التحتية

البندالقيمة
إطار العملABP 10.3 + .NET 10
ORMEntity Framework Core
قاعدة البيانات (محلي)SQL Server / LocalDB
قاعدة البيانات (إنتاج)PostgreSQL على Render
المصادقةOpenIddict + جلسة مخصصة HotelAuth
API URL (إنتاج)https://hotel-api-fo0z.onrender.com
Swagger/swagger على HttpApi.Host
