*مشروع مضياف العرب للفنادق v2.3.3*

---

## 1. نظرة عامة

عند الضغط على **«تحديث قاعدة البيانات»** في لوحة إعدادات DB، تمر العملية عبر:

1. **الواجهة (Angular)** — زر → خدمة HTTP → طلب PUT
2. **الشبكة** — proxy محلي أو Cloudflare Worker
3. **الخادم (ASP.NET + ABP)** — App Service → Migrations → Seed
4. **قاعدة البيانات** — PostgreSQL (إنتاج) أو SQL Server (محلي)

**العملية = خطوتان على الخادم:**
- **Migrations** — تحديث هيكل الجداول والأعمدة
- **Seed** — إكمال البيانات الافتراضية الناقصة

### 2.1 فتح اللوحة

**الدالة:** `toggleDbPanel(event)` — تعيين `dbPanelOpen = true/false`

### 2.2 لوحة إعدادات قاعدة البيانات

| الملف | الدور |
|-------|-------|
| `angular/src/app/shared/db-settings-panel/db-settings-panel.component.html` | زر «تحديث قاعدة البيانات» + spinner + رسالة الحالة |
| `angular/src/app/shared/db-settings-panel/db-settings-panel.component.ts` | منطق `updateDatabase()` |
| `angular/src/app/shared/db-settings-panel/db-settings-panel.component.css` | تنسيق اللوحة الجانبية |

**عند الضغط على التحديث:**
1. `updateLoading = true` — إظهار مؤشر التحميل
2. استدعاء `this.dbAdmin.updateDatabase()`
3. عند النجاح: عرض `result.message` في `db-panel__status`
4. عند الخطأ: `resolveOpError()` يترجم أخطاء HTTP (404, 405, JSON…)

### 2.3 خدمة HTTP

| الملف | الدور |
|-------|-------|
| `angular/src/app/services/hotel-database-admin.service.ts` | يرسل الطلب للـ API |

**الطلب:**

```
PUT {API_URL}/api/app/hotel-database-admin/database
Body: {}
```

**الاستجابة:**

```json
{ "success": true, "message": "تم تحديث قاعدة البيانات..." }
```

**زر النسخ الاحتياطي (مرتبط):**

```
POST {API_URL}/api/app/hotel-database-admin/backup
```

| الملف | الدور |

**UpdateDatabaseAsync() — خطوتان:**

```csharp
// 1) تطبيق Migrations

    await migrator.MigrateAsync();
```


1. EF Core يقرأ جدول `__EFMigrationsHistory`

## 8. التسلسل الزمني (خطوة بخطوة)

```
1. المستخدم يضغط زر الحساب → toggleDbPanel()
2. تفتح لوحة db-settings-panel
3. يضغط «تحديث قاعدة البيانات»
4. updateDatabase() → updateLoading = true
5. PUT /api/app/hotel-database-admin/database
6. proxy (محلي) أو worker.ts (إنتاج) → Render API
7. HotelDatabaseAdminAppService.UpdateDatabaseAsync()
8. EntityFrameworkCoreHotelDbSchemaMigrator.MigrateAsync()
9. EF Core يطبّق Migrations على PostgreSQL
10. IDataSeeder → كل DataSeedContributor
11. JSON { success, message } → Angular
12. عرض الرسالة في db-panel__status
```

## 9. ماذا يحدث لبياناتك؟

| العملية | التأثير |
|---------|---------|
| Migrations | يضيف/يعدّل أعمدة وجداول — لا يحذف الغرف أو الحجوزات |
| Seed | يكمّل الناقص فقط |
| لا يوجد | DROP TABLE، مسح حجوزات، استبدال كامل |

**نصيحة:** قبل التحديث على الإنتاج → «إنشاء نسخة احتياطية» (POST `/backup`).
