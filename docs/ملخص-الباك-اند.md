# ملخص الباك-اند التفصيلي — مضياف العرب للفنادق

*Modiaf.Al.Arab.Hotel — الإصدار v2.3.3 — ASP.NET Core + ABP Framework 10*

---

## 1. نظرة معمارية

الخادم مبني على **Domain Driven Design (DDD)** ضمن **ABP Framework 10**، ويتكون من طبقات منفصلة:

```
HttpApi.Host  →  HttpApi  →  Application  →  Domain  ←  EntityFrameworkCore
                    ↑              ↑
            Application.Contracts  Domain.Shared
```

| الطبقة | المشروع | الدور |
|--------|---------|-------|
| **Domain.Shared** | `Modiaf.Al.Arab.Hotel.Domain.Shared` | ثوابت، أدوار، توطين ABP |
| **Domain** | `Modiaf.Al.Arab.Hotel.Domain` | كيانات، مستودعات (واجهات)، بذور البيانات |
| **Application.Contracts** | `Modiaf.Al.Arab.Hotel.Application.Contracts` | DTOs، واجهات App Services |
| **Application** | `Modiaf.Al.Arab.Hotel.Application` | منطق التطبيق، App Services |
| **EntityFrameworkCore** | `Modiaf.Al.Arab.Hotel.EntityFrameworkCore` | DbContext، Repositories، Migrations |
| **HttpApi** | `Modiaf.Al.Arab.Hotel.HttpApi` | Controllers إضافية |
| **HttpApi.Host** | `Modiaf.Al.Arab.Hotel.HttpApi.Host` | نقطة التشغيل، CORS، Swagger، OpenIddict |
| **HttpApi.Client** | `Modiaf.Al.Arab.Hotel.HttpApi.Client` | عميل API للاختبار |
| **DbMigrator** | `Modiaf.Al.Arab.Hotel.DbMigrator` | ترحيل قاعدة البيانات |

### البنية التحتية

| البند | القيمة |
|-------|--------|
| **إطار العمل** | ABP 10.3 + .NET 10 |
| **ORM** | Entity Framework Core |
| **قاعدة البيانات (محلي)** | SQL Server / LocalDB |
| **قاعدة البيانات (إنتاج)** | PostgreSQL على Render |
| **المصادقة** | OpenIddict + جلسة مخصصة `HotelAuth` |
| **API URL (إنتاج)** | `https://hotel-api-fo0z.onrender.com` |
| **Swagger** | `/swagger` على HttpApi.Host |

## 3. طبقة Application — خدمات التطبيق

ABP ينشئ **REST API تلقائياً** من App Services عبر **Conventional Controllers**  
المسار الأساسي: `/api/app/{service-name}/{action}`

### 3.1 RoomAppService

**الملف:** `Application/Entities/Rooms.cs`  
**الواجهة:** `IRoomAppService`

| العملية | النوع | الوصف |
|---------|-------|-------|
| `GetListAsync` | GET | قائمة الغرف (CRUD) |
| `GetAsync` | GET | غرفة واحدة |
| `CreateAsync` | POST | إنشاء غرفة |
| `UpdateAsync` | PUT | تعديل غرفة |
| `DeleteAsync` | DELETE | حذف غرفة |
| `ResetAllStatusesAsync` | POST | إعادة تعيين حالات الغرف |

**ResetAllStatusesAsync:** يغيّر حالة كل الغرف إلى `TargetStatus` مع خيار `PreserveMaintenanceAndSuspended`.

### 3.2 BookingAppService

**الملف:** `Application/Entities/Bookings.cs`

- CRUD كامل للحجوزات
- تطبيع `Booking_Source` (direct/electronic/company/institution/employee)
- تطبيق العملة الافتراضية YER/YR

### 3.3 FloorAppService

**الملف:** `Application/Entities/Floors.cs` — CRUD للطوابق.

### 3.4 GuestRegistryAppService

**الملف:** `Application/Entities/GuestRegistries.cs`

| العملية | الوصف |
|---------|-------|
| CRUD | إنشاء/تعديل/حذف/عرض |
| `SaveProfileAsync` | حفظ ذكي: تحديث إن وُجد `Id_Number` |

### 3.5 IdentityTypeAppService / RoomTypeAppService

**الملفات:** `Application/Entities/IdentityTypes.cs`, `RoomTypes.cs` — CRUD.

### 3.6 HotelAuthAppService

**الملف:** `Application/HotelAuth/HotelAuthAppService.cs`

### 3.7 HotelAppUserAppService
