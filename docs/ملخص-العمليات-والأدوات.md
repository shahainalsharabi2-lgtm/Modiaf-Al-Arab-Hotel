# ملخص العمليات والأدوات — مشروع مضياف العرب للفنادق

*الإصدار: v2.3.3 — تاريخ التوليد: يونيو 2026*

---

## 1. نظرة عامة على المشروع

| البند | التفاصيل |
|-------|----------|
| **اسم المشروع** | Modiaf.Al.Arab.Hotel — مضياف العرب للفنادق |
| **النوع** | نظام إدارة فندق (PMS) |
| **الواجهة** | Angular 21 — RTL — متعدد اللغات |
| **الخادم (API)** | ASP.NET Core + ABP Framework 10 |
| **قاعدة البيانات** | PostgreSQL (إنتاج) / SQL Server (تطوير محلي) |
| **نشر الواجهة** | Cloudflare Workers — `hotel-management.shahainalsharabi2.workers.dev` |
| **نشر الـ API** | Render — `hotel-api-fo0z.onrender.com` |

## 2. ملخص العمليات (سير العمل)

### 2.1 تسجيل الدخول والجلسة

| العملية | المسار | الوصف |
|---------|--------|-------|
| تسجيل الدخول | `/login` | مصادقة المستخدم عبر `HotelAuthService` |
| حماية الصفحات | `authGuard` | منع الوصول للصفحات الداخلية بدون جلسة |
| إدارة الحساب | `/my-account` | تعديل بيانات المستخدم وكلمة المرور |

### 2.2 لوحة التحكم

| العملية | المسار | الوصف |
|---------|--------|-------|
| لوحة التحكم | `/dashboard` | ملخص الإشغال، بطاقات الحجوزات، وضع المطوّر |
| البحث العام | شريط علوي | اختصار F — بحث في الصفحات والعمليات |
| تغيير اللغة | شريط علوي | العربية + لغات إضافية (فرنسي، تركي، إندونيسي، صيني…) |

### 2.3 إدارة الغرف

| العملية | المسار | الوصف |
|---------|--------|-------|
| مخطط الغرف | `/rooms` | عرض شبكي للغرف حسب الطابق والحالة |
| شريط حالات العرض | يسار الصفحة | تصفية: متاحة ونظيفة، محجوزة، مغادرة، صيانة… |
| إضافة غرفة | `/rooms/add` | نموذج إنشاء غرفة جديدة |
| تعديل غرفة | `/rooms/edit/:id` | تعديل بيانات الغرفة |
| تفاصيل الغرفة | `/rooms/details/:id` | عرض تفصيلي للغرفة |
| غرفة فارغة (⋮) | قائمة البطاقة | حجز، تسكين مباشر، معاينة الغرفة |
| معاينة الغرفة | `room-preview-sheet` | فئة الغرفة، الأسرة، الموقع، المميزات |

**حالات الغرف:** متاحة، متسخة، مشغولة، محجوزة، صيانة، موقوفة، مغادرة متوقعة، خروج اليوم، محجوز غير نظيف، منقولة.

### 2.4 الحجوزات والاستقبال

| العملية | المسار | الوصف |
|---------|--------|-------|
| نموذج الحجز | `/booking` | إنشاء حجز أو تسكين (walk-in) أو نقل غرفة |
| تسكين مباشر | `/booking?mode=checkIn&walkIn=1` | تسكين ضيف بدون حجز مسبق |
| حجز من غرفة فارغة | `/booking?roomNumber=…` | حجز مرتبط برقم الغرفة |
| قائمة الحجوزات | `/bookings` | عرض PMS ببطاقات الحجوزات |
| الاستقبال | `/front-desk` | نفس واجهة الحجوزات (PMS) |
| بطاقة الحجز | `pms-booking-card` | تسكين، خروج، دفع، تفاصيل الضيف |

### 2.5 التقارير

| العملية | المسار | الوصف |
|---------|--------|-------|
| التقارير | `/reports` | تقارير الإشغال والإيرادات والإحصائيات |

### 2.6 الإعدادات

| التبويب | المفتاح | العمليات |
|---------|---------|----------|
| عام | `general` | اسم الفندق، الشعار، بيانات الاتصال |
| تقسيم الفندق | `layout` | طوابق، غرف، شبكة، إنشاء سريع لسلسلة غرف |
| طرق الدفع | `payments` | إضافة/تعديل/حذف طرق الدفع |
| أنواع الهوية | `identities` | إدارة أنواع وثائق الضيوف |
| الضيوف | `guests` | سجل الضيوف والحجوزات المرتبطة |
| العملة | `currency` | تحديد تلقائي للعملة حسب اللغة/فئة العربية |
| فئة اللغة العربية | `arabicLocale` | اختيار المنطقة والعملة للعربية |
| المستخدمون | `users` | إدارة مستخدمي النظام (مدير فقط) |
| المدخلات | `translations` | أكواد عامة (General Codes) |
| ترجمات الواجهة | `uiTranslations` | تحرير نصوص الشاشات بلغات متعددة |

### 2.7 المدخلات (General Codes)

| الفئة | المعرف | الاستخدام |
|-------|--------|-----------|
| أغراض الإقامة | `purposes-of-stay` | سبب زيارة الضيف |
| الجنسيات | `nationalities` | جنسية الضيف |
| صلة القرابة | `relationship-types` | علاقة المرافقين |
| أنواع الهوية | `identification-types` | وثائق التعريف |
| نوع التفضيل | `preference-type` | تصنيف التفضيلات |
| فئة اللغة العربية | `preference-category` | منطقة + عملة + ترتيب |
| فئات الغرف | `room-classes` | ديلوكس، سويت… |
| معمار الغرفة | `room-architecture` | شكل/بنية الغرفة |
| مميزات الغرف | `room-features` | Wi-Fi، مini bar… |
| مواقع الغرف | `room-locations` | موقع داخل المبنى |
| إطلالات الغرف | `room-views` | بحر، جبل… |
| أسباب الصيانة | `room-maintenance-reasons` | سبب إيقاف الغرفة |
| أسباب النقل | `room-move-reasons` | سبب نقل الضيف |

### 2.8 قاعدة البيانات والصيانة

|---------|--------|-------|

## 5. أدوات التطوير والنشر

### 5.1 أوامر الواجهة (Angular)

| الأمر | الوظيفة |
|-------|---------|
| `npm start` | تشغيل محلي HTTPS مع proxy للـ API |
| `npm run build:prod` | بناء الإنتاج |
| `npx wrangler deploy` | نشر على Cloudflare Workers |
| `npm run ui-translations:split` | تقسيم ملفات الترجمة |
| `npm run ui-translations:merge` | دمج الترجمات |
| `npm run ui-translations:sync` | مزامنة هيكل الترجمات |

### 5.2 أوامر الخادم (ASP.NET)

| الأمر | الوظيفة |
|-------|---------|
| `dotnet run` (HttpApi.Host) | تشغيل API محلياً |
| `dotnet run` (DbMigrator) | ترحيل قاعدة البيانات |
| `abp install-libs` | تثبيت مكتبات ABP |

### 5.3 أدوات البنية التحتية

| الأداة | الاستخدام |
|--------|-----------|
| **Cloudflare Wrangler** | نشر SPA على Workers |
| **Render** | استضافة API (Docker) |
| **PostgreSQL** | قاعدة بيانات الإنتاج |
| **OpenIddict** | مصادقة OAuth/OpenID |
| **Entity Framework Core** | ORM و Migrations |
| **ABP Framework 10** | إطار العمل الخلفي |

### 5.4 التقنيات الأساسية

| الفئة | الأدوات |
|-------|---------|
| Frontend | Angular 21, TypeScript 5.9, RxJS, Font Awesome |
| Backend | .NET 10, C#, ABP 10.3 |
| CSS | CSS مخصص + Bootstrap Icons |
| i18n | JSON (`ar.json`, `fr.json`, `tr.json`, `id.json`, `zh-Hans.json`) |
| اختبار | Vitest (Angular), xUnit (Backend) |
